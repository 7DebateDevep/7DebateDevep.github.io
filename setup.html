成都七中国际部辩论社区新成员准备流程

欢迎来到2020成都七中国际部辩论社区 !! 为了更好的适应社区活动安排, 请按照顺序完成以下检查清单

  • 注册社区邮箱（教程见下方, 请务必优先完成邮箱注册）

  • 注册社区论坛（教程见下方）

  • 注册社区学习系统（教程见下方）

  • 注册社区VPN系统（教程见下方）

  • 熟悉社区活动时间 http://www.7debate.club/schedule.html

  • 阅读并了解社区管理条例 http://www.7debate.club/regulation.pdf

  • 在学习系统中完成小测验

     

账户

  • 您在社区所有系统（邮箱，论坛，学习系统，VPN等）的用户名必须使用以下格式

    • 格式：名字全小写拼音 + “-” + 姓氏全小写拼音
    • 例子（张三）：san-zhang
    • 例子（李小明）：xiaoming-li

  • 若不确定是否填写正确请加入临时QQ群(群号: 655927392) 错误填写会导致无法参加社区任何活动!!!

 

社区邮件

社区邮件将用于所有学习系统的注册，官方比赛注册，以及社区正式公告的通知

  • 打开网页 https://exmail.qq.com/login

  • 选择手机号登录 （如果手机版不支持手机号登录，请使用电脑）

  • 使用在社区申请表上填写的手机号接收验证码并激活账号

  • 进入邮箱以后点击页面顶设置 - 账户

    • 更改账户昵称为你的名字拼音

      • 格式：名字全小写拼音 + “-” + 姓氏全小写拼音
      • 例子（张三）：san-zhang

    • 修改密码

  • 完成上述设置以后你的社区邮箱就成功激活了

    • 你的社区邮箱地址是名字拼音@7debate.club
    • 例子（李小明）：xiaoming-li@7debate.club

  • 社区所有在线平台全部使用社区邮箱，请务必保证邮箱设置准确无误

  • 如果遇到任何问题，通过加入临时QQ群(群号: 655927392)

     

社区论坛

所有辩题讨论，观点交流，以及其他任何的学术生活娱乐交流的线上平台。辩论社区从本学期开始不再使用QQ或者微信等社交平台

  • 设置完社区邮箱以后在邮箱里会有来自社区论坛的邀请邮件

  • 点击邀请链接开始账号注册（注意只有通过邀请链接才能激活注册）

    • username请填写名字的拼音

      • 格式：名字全小写拼音 + “-” + 姓氏全小写拼音
      • 例子（张三）：san-zhang

    • name请填写你的中文名字

  • 填写密码以后即可登录社区论坛

    • 此后请通过http://forum.7debate.club访问社区论坛

  • 修改头像

    • 点击右上角的默认头像 - 点击你的用户名 - 选择preference

    • 修改头像

      • 单人正脸，能让别人认识你即可

  • 如果遇到任何问题，通过加入临时QQ群(群号: 655927392)

     

社区学习系统

每周的作业，模拟辩论安排，以及讨论课作业都将在学习系统上进行

  • 访问网站https://www.gradescope.com/

  • 点击sign up 然后选择student

    • COURSE ENTRY CODE请填写 9N7GB4

    • Name请填写名字拼音

      • 格式：名字全小写拼音 + “-” + 姓氏全小写拼音
      • 例子（张三）：san-zhang

    • Email Address 请填写社区邮箱地址

    • STUDENT ID请留空

  • 注册登录以后如果能看到一门课程代号为Debate 100的课程，即操作成功

  • 如果遇到任何问题，请在社区论坛发帖提问

 

社区VPN系统

为社区所有成员提供访问Google, Youtube等国外网站的通道

  • 如果有自己VPN的同学可以不做这个部分

  • 打开网页http://vpn.7debate.club/sign-in

    • 选择sign up now
    • Email 请填写社区邮箱
    • Password 和 Confirm password自己设置即可
    • 点击 GET VERIFICATION CODE并在社区邮箱中查看

  • 注册完成后根据页面的指引完成服务器链接与软件下载

  • 如果遇到任何问题，请在社区论坛发帖提问
